# dLocal Basic Example

Minimal runnable dLocal charge example.

## Setup

1. Install dependencies:
   - `bun install`
2. Set environment variables:
   - `DLOCAL_X_LOGIN`
   - `DLOCAL_X_TRANS_KEY`
   - `DLOCAL_SECRET_KEY`

## Run

```bash
DLOCAL_X_LOGIN=login DLOCAL_X_TRANS_KEY=key DLOCAL_SECRET_KEY=secret bun run start
```
